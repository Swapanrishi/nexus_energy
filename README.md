
# Nexus IoT: Unified Management Hub

A world-class IoT dashboard featuring real-time telemetry, device lifecycle management, and Gemini-powered smart diagnostics.

## Features
- **Real-time Telemetry**: Simulated MQTT data stream for live monitoring.
- **Smart Diagnostics**: Integrated Google Gemini API for automated fleet analysis.
- **Device Management**: Full CRUD-like interface for managing edge nodes.
- **Responsive Design**: Built with Tailwind CSS for mobile and desktop support.

## Tech Stack
- **Frontend**: React 19, Tailwind CSS, Lucide React, Recharts.
- **Intelligence**: Google Gemini API (@google/genai).
- **Architecture**: Modular Service/Component pattern.

## Getting Started
1. Clone the repository.
2. Ensure you have your `API_KEY` set for Gemini functionality.
3. Open `index.html` or use a development server like Vite/ESBuild.
